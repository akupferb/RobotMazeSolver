TrackedRobot::TrackedRobot ( Maze *  maze)

This is a 1 args constructor.

This constructor prompts the user to enter the robot starting position. It then checks that the position is a valid input for the maze, updates the start and current position of the robot, and defines the robot name.

void TrackedRobot::update ( std::string  action)
virtual

This function pushes the current robot state to the top of the stack.

Parameters
actionA string of the direction of the robot action
Returns
None

For the chosen of the four actions, the function creates a pointer to the respective robot state and then updates the robot stack with that state. Then the pointer is deleted to free up the space.
